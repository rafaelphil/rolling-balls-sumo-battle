# Rolling Balls Sumo Battle

A fast paced local 1v1 game where two players battle to knock each other off the arena.

### Gameplay
Every time a player falls off the map, the other player gains a point and every players' position is reset back on the map. The game ends when one player reachs 7 points.

### Controls
The WASD keys control the blue ball and the arrow keys control the red ball.

### Running the game
You can play the game through your browser on itch.io
